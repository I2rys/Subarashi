## Installation
Github:

    git clone https://github.com/Subarashi-Project/Subarashi

NPM Packages:

    npm i readline && npm i request && npm i delay && npm i chalk && npm i json-hood && npm i secure-random-password && npm i puppeteer
    
## Usage

    node index.js
